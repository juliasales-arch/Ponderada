7 · Mapa de Interações
01
Cursor Personalizado
O cursor padrão do navegador é substituído por um ponto ciano com glow e um anel que segue com leve delay (lerp). Ao passar sobre elementos interativos (cards, botões, ícones), o anel expande de 32px para 48px com transição suave.
JavaScript · requestAnimationFrame · CSS transition
02
Navbar Flutuante com Highlight Ativo
Barra de navegação sticky com visual pill e backdrop blur. Ao scrollar, o IntersectionObserver detecta qual seção está visível e automaticamente destaca o botão correspondente na navbar com estilo "active" (fundo gradiente azul + glow). No mobile, vira um menu dropdown hamburguer.
JavaScript · IntersectionObserver · CSS sticky · backdrop-filter
03
Tooltip nos Ícones de Software
Ao passar o mouse sobre os ícones de software (Ps, Ai, Figma, Ae...), aparece uma tooltip acima do ícone com o nome completo do software, com animação de fade + slide. Implementado 100% em CSS puro usando o atributo data-tooltip e pseudo-elemento ::after.
CSS puro · ::after · attr() · opacity transition
04
Barras de Skill com Animação ao Scroll
As barras de habilidade começam em scaleX(0) e só animam quando entram na viewport do usuário, usando IntersectionObserver. Isso cria um efeito de "revelação progressiva" que valoriza a leitura e evita animações desperdiçadas fora da tela.
JavaScript · IntersectionObserver · CSS transform scaleX · cubic-bezier
05
Modal de Detalhes do Projeto
Clicar em qualquer project card abre um modal centralizado com overlay escuro e blur. O modal exibe nome, categoria, thumbnail e descrição completa do projeto. Fecha ao clicar fora, no botão ✕ ou pressionando Escape. Suporta imagens reais via tag img.
JavaScript · CSS transform + opacity · keyboard event · scroll lock
